What is Git?

  • Distributed Version Control
  • Advocates rapid branching and merging
  • Collaboration

Distributed Version Control

Everyone gets a copy!

  • Cloning a project gives you a full copy (commits, history, etc...)
  • Reliable like a trusty hammer
  • Fast like a racecar

Collaboration

  • Through remote repositories
  • Through visual tools (GitHub)

Try Git

Installing Git

  • Windows - Installer (Beware the whitelisting!)
  • Mac - Installer or through Homebrew
  • Linux - apt-get install git

Git/GitHub Strategies

Gatekeeper

  • Individual/small group manages the repository
  • Fork the project
  • Make branch, commit code
  • Send pull request
  • Gatekeeper approves or rejects

Our Approach

  • Everyone has read/write access
  • Make a branch, commit code
  • Send pull request
  • Another user reviews and approves or rejects
